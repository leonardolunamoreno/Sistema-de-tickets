# Sistema de tickets
Está es una versión inicial de un sistema de tickets, de momento el programa muestra una serie de opciones en la terminal, y también, todas las opciones con las que puede interactuar el usuario, son únicamente con la terminal.
El sistema guarda cada uno de los tickets creados por el usuario. 
El sistema puede mostrar cada uno de los tickets creados por el usuario. 
El sistema también muestra documentación sobre 2 problemas en partícular: Wi-fi lento y computadora que está funcionando lento, así como posibles soluciones ante estos problemas.
